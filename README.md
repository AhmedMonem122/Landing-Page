## Description
This project is built on using HTML, CSS and JavaScript(ES6 - ES 2015).
It implements DOM manipulation to toggle active classes on HTML elements.

## Instructions

To start the application :
1-Open your browser .
2-Press `ctrl+O` .
3-Navigate to your index.html .
4-Click on it .
